ZenAi is a full-stack AI-powered productivity suite built with the **PERN stack**. It provides a seamless experience for generating content, manipulating images, and analyzing documents through a unified, "zen" interface.

## üöÄ Features

* **Content Generation:** Create full-length articles and catchy blog titles using OpenAI.
* **Image Studio:** Generate high-quality images from text prompts.

* **Resume Reviewer:** Upload PDFs to get instant, AI-driven feedback on your CV.
* **Secure Auth:** Robust user management and authentication powered by Clerk.
